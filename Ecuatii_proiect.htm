Ecuații proiect

 

 

 

Condensatorul C se încarcă de la sursa E prin rezistența R conform ecuației diferențiale:

𝐸 = 𝑅 ⋅ 𝑖 + 𝑣𝑐 = 𝑅𝐶   + 𝑣𝑐

Această ecuație are soluția în domeniul timp:

𝑣𝑐 (𝑡) = 𝑉𝑐 (∞) − [𝑉𝑐 (∞) − 𝑉𝑐 (0)]

unde: 

 Vc(∞)  valoarea tensiunii la care s-ar încărca condensatorul, dacă ar

avea la dispoziție un timp nelimitat;

 Vc(0)  valoarea tensiunii de pe condensator, în momentul inițial;

Știind că, curentul prin condensator este de forma: iC = C   putem să aplicăm teoremele lui Kirckoff pe ochiul de circuit, rezultând un set de ecuații de forma:

  =   atunci când condensatorul se încarcă

= −  atunci când condensatorul se descarcă

Pentru diferențiatorul RC, ieșirea este egală cu tensiunea pe rezistor, adică: VOUT este egală cu VR și fiind o rezistență, tensiunea de ieșire se poate schimba instantaneu.

VOUT = VR = iR  R

iC = C

iR = iC

VOUT = RC

Atunci, putem vedea că tensiunea de ieșire VOUT este derivata tensiunii de intrare VIN care este ponderată cu constanta RC, unde RC reprezintă constanta de timp τ a circuitului serie.
